# 开发者快速开始

Haze 是企业内部的 Skill / MCP 能力广场，用于沉淀、审核、部署和共享团队内部可复用能力。

目前支持两类能力：

- Skill：面向提示词、流程说明、知识文档、操作规范等轻量能力。
- MCP：面向可调用工具、服务接口、自动化任务等可执行能力。

## 基本流程

1. 进入「开发者中心」
2. 点击「创建能力」
3. 选择能力类型：Skill 或 MCP
4. 填写基础信息：名称、标识、分类、简介、标签
5. 上传 ZIP 包
6. 保存草稿
7. 提交审核
8. 审核通过后进行部署和调试
9. 调试通过后发布到能力市场

## 发布后效果

能力发布后，企业内部用户可以在能力市场中查看、搜索和使用该能力。
